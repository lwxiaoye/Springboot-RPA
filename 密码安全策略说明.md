# 密码安全策略说明

## 功能概述

系统已实现完整的密码安全管理机制，包括：
- ✅ 密码 90 天强制过期
- ✅ 密码复杂度要求（字母 + 数字 + 符号）
- ✅ 密码长度限制（8-24 位）
- ✅ 登录时密码过期检测
- ✅ 强制修改过期密码

## 密码策略详情

### 1. 密码复杂度要求
新密码必须同时满足以下条件：
- ✓ 包含至少一个字母（a-z, A-Z）
- ✓ 包含至少一个数字（0-9）
- ✓ 包含至少一个特殊字符（如 !@#$%^&* 等）
- ✓ 长度在 8-24 位之间

### 2. 密码有效期
- **有效期**：90 天
- **过期检测**：每次登录时自动检测
- **过期处理**：登录后强制跳转到密码修改页面

### 3. 修改密码流程
1. 用户登录后，点击"个人信息"
2. 切换到"修改密码"标签页
3. 输入原密码和新密码
4. 系统自动校验密码复杂度
5. 修改成功后重新登录

## 数据库更新

### 方式一：新数据库（推荐）
直接运行 `init-db.sql` 创建全新的数据库：
```bash
mysql -u root -p < src/main/resources/init-db.sql
```

### 方式二：现有数据库升级
运行 `update-password-policy.sql` 添加新字段：
```bash
mysql -u root -p rpa_system < src/main/resources/update-password-policy.sql
```

## 用户使用指南

### 首次登录
1. 访问系统登录页面
2. 使用管理员账号登录：
   - 用户名：admin
   - 密码：admin123
3. 系统会提示修改密码（因为初始密码不符合新策略）

### 修改密码步骤
1. 登录后进入"个人信息"页面
2. 点击"修改密码"标签
3. 按照以下格式输入：
   - 原密码：当前使用的密码
   - 新密码：符合复杂度要求的密码（示例：Admin@123）
   - 确认密码：再次输入新密码
4. 点击"保存"

### 密码过期提醒
- 登录时如果密码已过期（90 天），系统会自动跳转到密码修改页面
- 必须在修改密码后才能正常使用系统

## 技术实现

### 后端改动
1. **User 实体**：新增 `passwordChangeTime` 字段
2. **UserService**：
   - `isPasswordExpired()` - 检查密码是否过期
   - `validatePasswordComplexity()` - 验证密码复杂度
   - `changePassword()` - 修改密码并更新时间戳
3. **UserController**：
   - `/api/user/check-password-expiry/{id}` - 检查密码过期接口
   - 登录接口返回 `passwordExpired` 标志

### 前端改动
1. **login.html**：
   - 登录成功后检查密码是否过期
   - 过期则跳转到 dashboard.html?changePassword=true
2. **admin.html**：
   - 新增密码复杂度校验
   - 新增密码过期检测
   - 强制显示密码修改 Tab

## 安全建议

1. **定期修改密码**：建议每 60 天主动修改一次
2. **密码示例**：
   - ✅ Admin@2024
   - ✅ Secure#Pass99
   - ✅ MyRpa@2024!
3. **避免使用**：
   - ❌ admin123（太简单）
   - ❌ password（常见密码）
   - ❌ 12345678（纯数字）
   - ❌ abcdefgh（纯字母）

## 常见问题

### Q: 忘记密码怎么办？
A: 联系系统管理员重置密码。管理员可以在"用户管理"中重置任何用户的密码。

### Q: 密码修改失败？
A: 检查新密码是否符合复杂度要求：
- 是否包含字母、数字、特殊字符
- 长度是否在 8-24 位之间

### Q: 如何查看密码何时过期？
A: 可以查询数据库：
```sql
SELECT username, password_change_time, 
       DATE_ADD(password_change_time, INTERVAL 90 DAY) as expire_date
FROM users;
```

### Q: 能否调整密码有效期？
A: 可以。修改 `UserService.isPasswordExpired()` 方法中的天数参数（默认 90 天）。

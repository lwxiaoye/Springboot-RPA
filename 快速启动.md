# RPA 系统 - 快速启动指南

## 重要提示

应用启动失败是因为数据库还未创建。请按以下步骤操作：

## 步骤一：创建数据库

### 方法 1：使用 MySQL 命令行工具

1. 打开命令行，登录 MySQL：
```bash
mysql -u root -p
```
（输入密码）

2. 执行以下 SQL 语句创建数据库：
```sql
CREATE DATABASE IF NOT EXISTS rpa_system DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

3. 退出 MySQL：
```bash
exit;
```

### 方法 2：使用 MySQL Workbench 或其他图形化工具

1. 连接到 MySQL 数据库
2. 右键点击 "Schemas" → "Create Schema..."
3. 输入名称：`rpa_system`
4. 字符集选择：`utf8mb4`
5. 点击 Apply

## 步骤二：修改数据库密码（可选）

如果您的 MySQL root 用户密码不是 `root`，需要修改配置文件：

编辑 `src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/rpa_system?useSSL=false&serverTimezone=UTC&characterEncoding=utf8
    username: root          # 你的数据库用户名
    password: YOUR_PASSWORD # 改为你的实际密码
```

## 步骤三：重新启动应用

```bash
mvn spring-boot:run
```

或者在 IDE 中运行 `Main.java`

## 步骤四：访问系统

浏览器打开：http://localhost:8080

### 默认测试账号
- 用户名：admin  密码：admin123
- 用户名：user1  密码：user123

## 常见问题

### 1. 不知道 MySQL 密码怎么办？

**Windows 系统**：
- 默认安装时密码可能是 `root` 或空
- 如果都不对，需要重置 MySQL 密码

**重置密码方法**：
1. 停止 MySQL 服务
2. 使用 `--skip-grant-tables` 参数启动
3. 执行 `ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';`
4. 重启 MySQL 服务

### 2. MySQL 服务未启动

**Windows**：
```bash
# 以管理员身份运行
net start MySQL80
```
（服务名可能是 MySQL、MySQL80、MySQL57 等）

### 3. 端口被占用

如果 8080 端口被占用，修改 `application.yml`：
```yaml
server:
  port: 8888  # 改为其他端口
```
